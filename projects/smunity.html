주요 구현 내용

  • 졸업요건 판정 로직 구현 — 이수 학점 총계, 필수 과목 이수 여부, 영어 인증, 교양 분류 등 학과 규정을 Python 로직으로 코드화. 재학생이 학번과 성적표를 입력하면 졸업 가능 여부를 즉시 판정
  • NGINX 리버스 프록시 구성 — 정적 파일 서빙을 Django에서 분리하여 NGINX에서 직접 처리. HTTPS 인증서 적용으로 보안 연결 제공
  • Docker Compose 배포 환경 — Django, NGINX, 데이터베이스 컨테이너를 docker-compose로 구성하여 개발·운영 환경 일관성 확보 및 배포 자동화
  • AWS EC2 장기 운영 — 약 3년간 실서비스를 운영하며 학과 학번별 커리큘럼 변경 사항을 직접 반영. 누적 방문자 10만 명 달성
  • 동기 협업 개발 — 컴퓨터과학과 19학번 동기들과 기획부터 운영까지 전 과정 협업. 코드 리뷰와 이슈 트래킹으로 서비스 품질 유지
